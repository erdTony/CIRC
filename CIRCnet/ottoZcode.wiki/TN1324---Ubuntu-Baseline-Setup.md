# TN1224 Ubuntu Baseline Setup

This will guide setup of base line Linux software for a new installation.

* [ ] Snap - ensure `snapd` is available
  * $ `sudo apt update`
  * $ `sudo apt install snapd`

* [ ] Chromium
  * Reference: [Linuxize](https://linuxize.com/post/how-to-install-chromium-web-browser-on-ubuntu-20-04/)
  * $ `sudo snap install chromium`

* [ ] WhatsApp
  * Reference: [Snap Craft](https://snapcraft.io/install/whatsapp-for-linux/ubuntu)
  * $ `sudo snap install whatsapp-for-linux`
